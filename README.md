CareerStart is a web-based job portal designed specifically for freshers and students to explore entry-level job and internship opportunities.

The application provides an easy-to-use interface where users can search and view job postings, create and manage their profiles, and apply for suitable roles. Recruiters can post job openings and review candidate applications through the platform.

The goal of CareerStart is to simplify the job search process for first-time job seekers by offering a clear, organized, and beginner-friendly experience.

## Technologies Used

- Frontend: HTML, CSS, JavaScript, React  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JSON Web Token (JWT)  
- Tools: Git, GitHub, VS Code
